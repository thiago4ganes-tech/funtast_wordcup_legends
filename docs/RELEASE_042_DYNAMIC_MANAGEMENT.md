# Release 0.4.2 — Gestão Tática Dinâmica

## Identidade e cache

O HTML mostra explicitamente:

```text
Release 0.4.2 — Gestão tática dinâmica
Build fwcl-0-4-2-20260707
```

Todos os arquivos CSS e JavaScript recebem `?v=0.4.2`.  
Os JSONs de produção usam a chave de build e `cache: no-store`.

## Mecânicas adicionadas

- banco automático de sete reservas;
- fadiga individual baseada em minutos, intensidade, função e stamina;
- substituições automáticas por desgaste, lesão, cartão e necessidade tática;
- limite de cinco substituições;
- mudanças entre 5-4-1, 4-5-1, 4-2-3-1, 4-3-3, 4-2-4, 3-4-3 e 3-3-4;
- perfis de árbitro equilibrado, rigoroso, permissivo e técnico;
- aplicação da vantagem;
- atendimentos físicos e substituições forçadas quando necessário;
- painel ao vivo com formação, postura, condição física e substituições;
- relatório final de gestão da partida.

## Observação

Os reservas do XI Fantasy são selecionados automaticamente entre jogadores não utilizados das seleções e edições sorteadas para o time titular. Não há cobrança adicional de orçamento nesta versão.
