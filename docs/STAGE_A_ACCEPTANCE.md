# Checklist de Aceite — WCHD Etapa A

A Etapa A só estará concluída quando todos os itens críticos forem atendidos.

## 1. Cobertura estrutural

```text
[ ] Todas as Copas de 1986 até 2022 cadastradas
[ ] Todas as seleções dessas Copas cadastradas
[ ] Todas as partidas dessas Copas cadastradas
[ ] Todos os placares conferidos
[ ] Todas as fases classificadas corretamente
```

## 2. Jogadores e elencos

```text
[ ] Todos os elencos vinculados às seleções
[ ] Todo jogador-copa vinculado a um atleta-base
[ ] Atletas com múltiplas Copas normalizados no mesmo player_base_id
[ ] Nomes alternativos registrados
[ ] Seleções com menos de 23 jogadores marcadas como incompletas
```

## 3. Eventos oficiais

```text
[ ] Gols vinculados às partidas
[ ] Gols vinculados aos jogadores, quando possível
[ ] Cartões vinculados aos jogadores
[ ] Substituições vinculadas às partidas
[ ] Pênaltis vinculados, quando disponíveis
```

## 4. Qualidade

```text
[ ] Todos os registros têm source_primary
[ ] Todos os registros têm source_confidence
[ ] IDs únicos sem duplicidade
[ ] Partidas sem times ausentes
[ ] Eventos sem match_id inexistente
[ ] Validação automática executada
```

## 5. Consultas mínimas suportadas

```text
[ ] Listar seleções por Copa
[ ] Listar jogadores por seleção-Copa
[ ] Listar partidas de uma seleção-Copa
[ ] Listar gols de um jogador-Copa
[ ] Listar cartões de um jogador-Copa
[ ] Listar jogadores que atuaram juntos
[ ] Identificar confrontos entre países
```
