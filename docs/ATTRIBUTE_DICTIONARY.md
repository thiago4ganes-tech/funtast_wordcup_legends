# Dicionário de Atributos — WCHD Etapa C

## 1. Atributos ofensivos

| Atributo | Descrição | Evidências típicas |
|---|---|---|
| finishing | Capacidade de converter chances | gols, gols/90, share de gols, gols em mata-mata |
| heading | Finalização pelo alto | gols de cabeça quando houver, função, altura, perfil aéreo coletivo |
| dribbling | Capacidade de superar marcação | dribles quando disponíveis, função tática, perfil de condução |
| vision | Capacidade de criar passes decisivos | assistências, participação criativa, função de meia/organizador |
| passing | Qualidade geral de passe | assistências, função, posse coletiva, minutos em meio-campo |
| crossing | Qualidade de cruzamento | assistências por cruzamento quando houver, laterais/pontas, perfil de time |
| pace | Capacidade de ruptura e transição | função, idade, perfil de transição, consenso histórico documentado |
| power | Força física/chute/duelo | posição, perfil físico, gols de média distância quando houver |
| off_ball_movement | Movimento sem bola | gols, titularidade, presença em área, participação em gols |

---

## 2. Atributos defensivos

| Atributo | Descrição | Evidências típicas |
|---|---|---|
| marking | Marcação individual/zonal | posição, defesa coletiva, minutos, função |
| tackling | Desarme/combate | ações defensivas recentes, função, cartões como contraponto |
| interception | Leitura defensiva | posição, defesa coletiva, função tática |
| aerial_defense | Defesa aérea | posição, altura, perfil defensivo, gols sofridos pelo alto quando houver |
| defensive_positioning | Posicionamento defensivo | defesa coletiva, minutos, função, estágio alcançado |

---

## 3. Goleiros

| Atributo | Descrição | Evidências típicas |
|---|---|---|
| reflexes | Reflexo em finalizações | gols sofridos, clean sheets, saves quando houver |
| handling | Segurança em bolas defensáveis | dados de goleiro quando houver, defesa coletiva |
| one_v_one | Saída em mano a mano | dados recentes, perfil histórico, consenso documentado |
| penalty_save | Defesa de pênaltis | pênaltis defendidos, disputas, histórico |
| aerial_claim | Saída em cruzamentos | altura, perfil, cruzamentos defendidos quando houver |

---

## 4. Mentais/contextuais

| Atributo | Descrição | Evidências típicas |
|---|---|---|
| decision | Tomada de decisão | assistências, baixa perda quando houver, função de criação |
| composure | Controle sob pressão | desempenho em mata-mata, pênaltis, finais |
| clutch | Impacto em momentos decisivos | gols/assistências em mata-mata, final, semifinal |
| leadership | Liderança estrutural | capitão, minutos, função, relevância no time |
| discipline | Risco disciplinar inverso | cartões, expulsões, faltas quando houver |
| reliability | Consistência e disponibilidade | minutos, titularidade, poucas punições |
| stamina | Capacidade de sustentar minutos | minutos, jogos completos, função física |

---

## 5. Escala

Todos os atributos usam escala de 1 a 99.

Faixas recomendadas:

| Faixa | Interpretação |
|---|---|
| 90–99 | Elite histórica |
| 80–89 | Muito forte |
| 70–79 | Bom |
| 60–69 | Regular |
| 50–59 | Fraco para o nível histórico |
| <50 | Uso raro, apenas para lacunas ou função incompatível |

---

## 6. Confiança

Cada atributo deve ter:

```text
confidence_level: A, B, C ou D
```

- A: evidência direta forte;
- B: evidência confiável, mas incompleta;
- C: estimativa com base parcial;
- D: provisório.
