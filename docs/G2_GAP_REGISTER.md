# Registro de Lacunas — G2

As lacunas detalhadas estão em:

```text
data/quality/g2_gap_register.json
data/quality/csv/g2_gap_register.csv
```

## Lacunas críticas

1. Assistências padronizadas.
2. Combinações diretas de gol entre jogadores.
3. Passes, passes-chave e passes progressivos.
4. Chutes e chutes no alvo por jogador.
5. Dribles, desarmes, interceptações e pressões.
6. Estilo coletivo das seleções.
7. Sinergia verdadeira por tracking ou evento direto.

## Diretriz para G3

A G3 deve preencher lacunas com a seguinte prioridade:

```text
1. dado real complementar
2. proxy estimado a partir de dado real
3. sintético comparável com limite de peso
4. manual review
5. manter missing quando não houver base razoável
```
