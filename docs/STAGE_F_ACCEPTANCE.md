# Checklist de Aceite — Etapa F

A Etapa F estará concluída quando:

## 1. Calibração

```text
[ ] calibration_targets definido
[ ] calibration_weights definido
[ ] quality gate estatístico definido
[ ] playbook de ajuste definido
```

## 2. Regressão

```text
[ ] matriz de fixtures definida
[ ] testes críticos definidos
[ ] testes de integridade definidos
[ ] testes visuais definidos
```

## 3. Release candidate

```text
[ ] manifest da release candidate criado
[ ] plano go/no-go criado
[ ] plano de rollback criado
[ ] release notes preliminar criada
```

## 4. Pronto para Release 0.4

```text
[ ] motor pode rodar em modo legacy
[ ] motor pode rodar em modo calibrated
[ ] modo comparison definido
[ ] quality gate decide aprovação/reprovação
[ ] nenhuma integração deve quebrar montagem de elenco
```
