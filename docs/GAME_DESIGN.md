# Game Design — Fantasy World Cup Legends

## Modos

### Dinastia Nacional
O jogador escolhe um país. Cada posição sorteia uma seleção histórica daquele país, por ano/copa. O usuário escolhe um jogador compatível daquela convocação.

### Fantasy Mundial
Cada posição sorteia qualquer seleção histórica da base WCHD. O usuário escolhe um jogador compatível daquela convocação.

## Regras de mercado

- O sorteio é vinculante.
- Não há resortear.
- Ao selecionar uma posição, o jogador deve escolher alguém antes de clicar em outra posição.
- O orçamento base em dificuldade normal é US$100MM.
- Não é permitido contratar jogador que ultrapasse o budget.
- Não é permitido repetir o mesmo atleta, mesmo em versões de Copas diferentes.

## Partida

A partida é simulada pelo LEGION Event Graph. O placar evolui com os eventos, e o relatório final explica o resultado com base nos eventos gerados.
