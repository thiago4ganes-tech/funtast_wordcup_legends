# Checklist de Aceite — Etapa D

A Etapa D estará concluída quando:

## 1. Arquitetura

```text
[ ] Event Graph organizado por eventos-mãe
[ ] Eventos terminais definidos
[ ] Fases do lance definidas
[ ] Match State definido
[ ] Memória anti-repetição definida
```

## 2. Probabilidades

```text
[ ] Modelo de probabilidade documentado
[ ] Modificadores ofensivos aplicados
[ ] Modificadores defensivos aplicados
[ ] Modificadores de goleiro aplicados
[ ] Contexto de placar aplicado
[ ] Contexto de minuto aplicado
[ ] Clamps aplicados
```

## 3. Estatísticas

```text
[ ] Gols nascem de finalizações
[ ] xG nasce dos eventos
[ ] Finalizações no alvo coerentes
[ ] Escanteios, faltas e cartões registrados
[ ] Participações de jogadores registradas
```

## 4. Narração

```text
[ ] Narração recebe dados do evento
[ ] Frases variam por tipo de lance
[ ] Gols têm alerta próprio
[ ] Relatório final usa estatísticas reais da simulação
```

## 5. Testes

```text
[ ] Confrontos fixos definidos
[ ] Simulação em lote preparada
[ ] Métricas médias calculadas
[ ] Sinais de problema documentados
[ ] Critério de aceite definido
```
