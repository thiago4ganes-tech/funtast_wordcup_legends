# Limitações e Próximos Passos — G4

## 1. Limitações

A G4 conclui a base operacional, mas ainda há limitações naturais:

- dados técnicos antigos não existem como tracking moderno;
- passes, dribles e pressões são sintéticos comparáveis;
- sinergia direta ainda depende de estimativa quando não há assistência observada;
- estilos coletivos são aproximados por resultados, gols, contexto e função.

## 2. Próximos passos

```text
1. Consolidar A-G em pacote único
2. Integrar wchd_legion_inputs.json na Release 0.4
3. Rodar quality gate de simulação
4. Ajustar engine_weight_limit conforme testes
5. Adicionar painel de qualidade de dados no jogo
```

## 3. Regra permanente

Toda melhoria futura de dados deve preservar:

```text
data_origin
confidence_level
source_basis
estimation_method
```
